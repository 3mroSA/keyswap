# KeySwap
A simple tool in C# that instantly converts selected text between keyboard layout

## Usage
Have you ever been typing while looking at your keyboard and not noticing that you were typing in a different language?
I made this program to fix this, select the text you want to convert, press CTRL + F8 and your text has been converted to the layout chosen.

## Customizability
You are able to change and add more layouts. Default layouts are english -> arabic and vice versa.

If you plan on makinag a layout, heres the template:
``1=2
2=3
4=5``
